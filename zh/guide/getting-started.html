        # 快速上手

        # 安装

        yarn add vuepress-theme-feed -D
# 或 npm install vuepress-theme-feed -D

        # 使用

        // .vuepress/config.js
module.exports = {
  theme: 'feed',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}

        # Feedpage

        在README.md中指明feed: true即可

        ---
feed: true
---
