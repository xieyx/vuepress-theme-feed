        # Getting Started

        # Install

        yarn add vuepress-theme-feed -D
# OR npm install vuepress-theme-feed -D

        # Usage

        // .vuepress/config.js
module.exports = {
  theme: 'feed',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}

        # Feed page

        To use it, specify feed: true in your README.md

        ---
feed: true
---
